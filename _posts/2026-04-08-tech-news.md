## 八、近期重要论文

### 🔥 本周热门（社区跨平台高讨论度）

- **LlamaFactory: Unified Efficient Fine-Tuning of 100+ Language Models** — HF 182↑
  统一的 LLM 微调框架，Web UI 一键覆盖 100+ 模型，正在成为社区微调事实标准。
  🔗 [HuggingFace Papers](https://huggingface.co/papers/trending)

- **VibeVoice — Microsoft Research** — HF 153↑
  长时多说话人语音合成，结合 next-token 扩散与高效连续语音 tokenizer，质量显著优于既有 TTS 基线。
  🔗 [HuggingFace Papers](https://huggingface.co/papers/trending)

- **MinerU2.5: Decoupled VLM for High-Resolution Document Parsing** — HF 155↑
  1.2B 参数文档解析 VLM，coarse-to-fine 策略下达成 SOTA，单页解析延迟显著降低。
  🔗 [HuggingFace Papers](https://huggingface.co/papers/trending)

- **SmolDocling — IBM Granite** — HF 156↑
  仅 256M 参数的端到端文档转换 VLM，覆盖多种版式文档，端侧部署友好。
  🔗 [HuggingFace Papers](https://huggingface.co/papers/trending)

- **PaddleOCR-VL — PaddlePaddle** — HF 123↑
  0.9B 多语言文档解析 VLM，结合 NaViT 动态分辨率与 ERNIE，多项基准刷新 SOTA。

### ⭐ 推理优化

- **TriAttention: Efficient Long Reasoning with Trigonometric KV Compression — NVIDIA** — HF 64↑
  通过 Q/K 向量集中性改进 key 重要性估计，显著缓解长上下文 KV cache 内存瓶颈，长推理任务收益明显。
  🔗 [HuggingFace Papers](https://huggingface.co/papers/trending)

- **Deep-Thinking Ratio (DTR) — Google Research**
  提出 Think@n 推理范式：在 AIME 25 上准确率高于标准多数投票的同时，**总推理成本下降 49%**。
  🔗 [MarkTechPost](https://www.marktechpost.com/2026/02/21/a-new-google-ai-research-proposes-deep-thinking-ratio-to-improve-llm-accuracy-while-cutting-total-inference-costs-by-half/)

- **Bitnet.cpp**
  ternary LLM 的混合精度 GEMM 库，边缘设备推理速度大幅领先现有方案。

### 🧠 架构与训练

- **Self-Distilled RLVR (RLSD)** — alphaXiv 热门
  RL + 自蒸馏的 reasoning 训练范式，相比 GRPO 在多模态基准上**绝对准确率 +2.32%**。
  🔗 [alphaXiv](https://www.alphaxiv.org/)

- **Embarrassingly Simple Self-Distillation Improves Code Generation**
  纯自生成解的 SSD 方法，把 Qwen3-30B-Instruct 在 LiveCodeBench v6 的 pass@1 从 **42.4% 拉到 55.3%**，无需外部 verifier。
  🔗 [alphaXiv](https://www.alphaxiv.org/)

- **The Latent Space: Foundation, Evolution, Mechanism, Ability, and Outlook** — HF 130↑
  系统性梳理 latent space 作为语言模型基础计算基底的角色与演化路径。

### 🤖 Agent

- **Agent Lightning: Train ANY AI Agents with Reinforcement Learning** — HF 140↑
  通用 Agent RL 框架，分层 RL 算法，将执行与训练解耦，对接异构 Agent 实现简洁。

- **SKILL0: In-Context Agentic RL for Skill Internalization** — HF 86↑
  动态课程让 LLM Agent 把技能内化到参数中，**ALFWorld 成功率 87.9%（+9.7%）**，同时显著降低 context token 成本。
  🔗 [alphaXiv](https://www.alphaxiv.org/)

- **OpenClaw-RL: Train Any Agent Simply by Talking** — HF 149↑
  通过自然语言指令对接异构交互模态的策略学习框架，结合 PRM judge 异步训练。

- **MetaClaw: Just Talk — Meta-Learning Agent in the Wild** — HF 136↑
  持续元学习 Agent，联合演化策略与可复用行为技能，最大化在线时间。

- **CORAL: Autonomous Multi-Agent Evolution for Open-Ended Discovery**
  数学与系统优化任务上 11 项中 8 项 SOTA。
  🔗 [alphaXiv](https://www.alphaxiv.org/)

- **Meta-Harness: End-to-End Optimization of Model Harnesses**
  端到端优化 LLM 外部代码接口，文本分类 +7.7 分、数学推理 +4.7 分。
  🔗 [alphaXiv](https://www.alphaxiv.org/)
